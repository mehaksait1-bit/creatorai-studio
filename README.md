# CreatorAI Studio

CreatorAI Studio is a web-based AI content generation platform.

## Features

* Script Generator
* Caption Generator
* Hashtag Generator
* Image Prompt Generator
* History Tracking
* Copy & Download Options

## Technologies Used

* React.js
* JavaScript
* Tailwind CSS
* Node.js
* Express.js
* LocalStorage
